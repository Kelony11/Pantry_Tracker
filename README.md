# Pantry Tracker 
# PROJECT OVERVIEW
This project is a website that keeps track of all your pantry items in an inventory.

# KEY FEATURES 🔑
- Special buttons to Add and Remove items to/from the pantry.
- Search bar to filter to a specific item.
